# ExCortex Seed Data Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers-extended-cc:executing-plans to implement this plan task-by-task.

**Goal:** Pre-populate ExCortex with opinionated clusters, neurons, ruminations, engrams, signals, senses, and axioms so the app is useful out of the box.

**Architecture:** A single `ExCortex.Seeds` module with an idempotent `seed/0` function. Each entity type has its own private function. Uses existing context modules (Clusters, Ruminations, Memory, Signals, Lexicon) and direct Repo.insert for Neurons/Senses. Called from `priv/repo/seeds.exs`.

**Tech Stack:** Elixir, Ecto, existing ExCortex context modules.

**Files:**

```elixir
# lib/ex_cortex/seeds.ex

  @moduledoc "Seeds the database with starter clusters, neurons, ruminations, engrams, signals, senses, and axioms."

  alias ExCortex.Lexicon

  alias ExCortex.Ruminations
  alias ExCortex.Ruminations.Synapse

  def seed do
    Logger.info("[Seeds] Seeding ExCortex...")
    seed_clusters()
    seed_neurons()
    seed_ruminations()
    seed_engrams()
    seed_axioms()
    seed_signals()
    seed_senses()
    Logger.info("[Seeds] Done.")
  end

  # Private functions added in subsequent tasks...
  defp seed_clusters, do: :ok
  defp seed_neurons, do: :ok
  defp seed_ruminations, do: :ok
  defp seed_engrams, do: :ok
  defp seed_axioms, do: :ok
  defp seed_signals, do: :ok
  defp seed_senses, do: :ok
end
```

```elixir
@clusters [
  {"Research", "Gather, synthesize, and extract insights from URLs, feeds, and documents. Use the gatherer to fetch raw content, the analyst to find patterns across sources, and the summarizer to distill key takeaways into engrams."},
  {"Writing", "Draft, edit, and tone-check prose. The drafter produces initial content, the editor refines structure and clarity, and the tone_checker ensures the voice matches the audience. Handles blog posts, documentation, and communications."},
  {"Ops/Infra", "Monitor system health, validate deployments, and audit dependencies. The monitor watches for anomalies, the auditor checks for outdated or vulnerable packages."},
  {"Triage", "Intake data from senses, classify by priority and type, and route to the appropriate cluster. The classifier determines urgency, the router dispatches."},
  {"Memory Curator", "Maintain memory hygiene. Periodically review engrams for duplicates, stale content, and gaps. Consolidate related memories, promote important ones to higher tiers, and ensure tags are consistent."},
  {"Daily Briefing", "Aggregate signals from all active senses into a concise morning summary. The aggregator collects overnight activity, the editor compresses it into a scannable briefing card on the cortex dashboard."},
  {"Learning", "Extract knowledge from articles, papers, and videos. The extractor pulls key concepts, the connector links them to existing engrams and identifies relationships."},
  {"Creative", "Brainstorming, ideation, and lateral thinking. The diverger generates many possibilities without judgment, the connector finds unexpected links between ideas. Divergent over convergent."},
  {"Devil's Advocate", "Stress-test proposals and plans. The critic finds weaknesses, missing edge cases, and false assumptions. The steelman articulates the strongest counter-argument to keep the critique honest."},
  {"Sentinel", "Watch for things that should have triggered action but haven't. Stale PRs, overdue TODOs, senses that stopped reporting, daydreams that failed silently. The watcher queries, the alerter surfaces."},
  {"Translator", "Convert between contexts. Technical explanation to plain English. Code to documentation. Slack thread to action items. Meeting notes to tickets."},
  {"Archivist", "Package engrams and signals into publishable artifacts. The collector gathers relevant memories, the packager formats them into changelogs, blog drafts, weekly reports, or project summaries."},
  {"Therapist", "Tone and sentiment analysis on inputs. Detect when conversations are getting heated, when PR reviews are unusually terse, when a thread needs de-escalation. Social signal processing."}
]

defp seed_clusters do
  Enum.each(@clusters, fn {name, pathway} ->
    case Clusters.upsert_pathway(name, pathway) do
      {:ok, _} -> Logger.info("[Seeds] Cluster: #{name}")
      {:error, e} -> Logger.warning("[Seeds] Failed cluster #{name}: #{inspect(e)}")
    end
  end)
end
```
